This folder contains data for various UI components.
- Arrow_Big.png: Large arrow drawn in the map image, for example.
- Arrow.png: Arrow used in menus.
- font_8x8.png: Default 8x8 font used for text.
- GUI.png: The default "window" skin.
- Map.png: A map graphic drawn when viewing the map screen.
- MapMarkerArrow.png: The arrow drawn to show where the player is.
- MapTiles.png: Tileset used for the minimap.
